# Python-pip
Curso Python: PIP y entornos virtuales

> Para correr el juego debes seguir las siguientes instrucciones en la terminal:

# Game project
```sh
cd game
python3 main.py
```


# App project
```sh
git clone
cd app
python3 -m venv vapp
source vapp/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
